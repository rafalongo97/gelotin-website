# Gelotin - Site Institucional Premium

Um site institucional moderno, minimalista e altamente profissional para a **Gelotin**, empresa produtora e distribuidora de gelo (em tubos e em escamas). O site conta com recursos focados em conversão, pureza da água, cálculo automático de pedidos e integração direta com o WhatsApp.

## ❄️ Funcionalidades

- **Design Premium e Minimalista**: Layout com efeito *glassmorphism* (simulando texturas e transparências do gelo), paleta de cores moderna (tons frios) e bastante espaço de respiro.
- **Sobre a Qualidade**: Seção dedicada a explicar os processos de purificação da água e durabilidade do gelo em conformidade com as normas sanitárias.
- **Apresentação de Produtos**: Destaque comercial para os dois produtos principais:
  - **Gelo em Tubos (3kg)**: Ideal para copos, drinks e consumo direto.
  - **Gelo em Escamas (10kg)**: Ideal para uso comercial, buffets e conservação de alimentos.
- **Calculadora de Consumo de Gelo**: Uma ferramenta interativa em JavaScript que calcula automaticamente a quantidade sugerida de sacos de gelo com base no tipo de evento e número de convidados.
- **Integração com WhatsApp**:
  - Envio direto de pedidos formatados a partir do resultado da calculadora.
  - Formulário de contato estruturado que dispara o texto do lead diretamente para o WhatsApp de vendas da empresa.
- **Localização e Mapa**: Integração de mapa interativo estilizado para combinar com o design institucional.
- **Responsividade Total**: Design desenvolvido seguindo a abordagem *mobile-first*, adaptando-se a dispositivos móveis, tablets e monitores desktop.

## 🛠️ Tecnologias Utilizadas

- **HTML5** (Estrutura semântica e SEO)
- **CSS3** (CSS Grid, Flexbox, Variáveis CSS, Glassmorphism, Micro-animações e Transitions)
- **JavaScript** (Manipulação de DOM, cálculos interativos e integração com API do WhatsApp)
- **Lucide Icons** (Ícones vetorizados minimalistas de alta performance)
- **Google Fonts** (Fontes Inter e Poppins)

## 🚀 Como Rodar o Projeto Localmente

O projeto é estático e super leve, não necessitando de builds ou instalações complexas. Para rodá-lo localmente:

1. **Abrir diretamente**:
   Basta abrir o arquivo `index.html` em qualquer navegador web.

2. **Usando o Servidor de Desenvolvimento Integrado (Node.js)**:
   Se você possui o Node.js instalado, pode rodar o servidor estático minimalista incluído:
   ```bash
   node server.js
   ```
   Depois, acesse em seu navegador: `http://localhost:3000`
